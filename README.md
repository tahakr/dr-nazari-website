# Dr. Mohammad Nazari - Professional Website

A modern, professional website for Dr. Mohammad Nazari, specialist in interventional radiology.

## 🚀 Getting Started

### Prerequisites
- Node.js 16.x or higher
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/tahakr/dr-nazari-website.git

# Install dependencies
cd dr-nazari-website
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the website.

## 🛠️ Build & Deploy

```bash
# Build for production
npm run build

# Start production server
npm run start
```

## 📦 Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Icons**: Lucide React
